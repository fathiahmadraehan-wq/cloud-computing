Pengertian

Pengertian Cloud Computing

Cloud computing adalah model pengiriman layanan komputasi melalui internet yang memungkinkan akses on-demand ke sumber daya komputasi dengan model pay-as-you-go.

Karakteristik 1

🌐 On-Demand

Kapasitas komputasi otomatis tanpa interaksi langsung penyedia.

Karakteristik 2

🔗 Broad Access

Tersedia via jaringan, diakses dari semua perangkat.

Karakteristik 3

🏊 Resource Pool

Sumber daya dikumpulkan untuk melayani banyak konsumen.

Karakteristik 4

⚡ Rapid Elasticity

Kapasitas diprovisikan elastis sesuai permintaan.

Karakteristik 5

📊 Measured Service

Penggunaan diukur otomatis — pay-as-you-go transparan.

Model Penerapan

Model Penerapan Cloud

Model penerapan menentukan di mana infrastruktur cloud dikelola dan siapa yang memiliki akses.

🌐

Public Cloud

Infrastruktur milik penyedia, dibagikan banyak org. Contoh: AWS, GCP, Azure.

✓ Biaya rendah   ✓ Skalabilitas tinggi
🏢

Private Cloud

Eksklusif satu organisasi. Kontrol dan keamanan penuh.

✓ Keamanan tinggi   ✓ Kontrol penuh
🔀

Hybrid Cloud

Kombinasi public & private cloud. Fleksibel dan mendukung kepatuhan.

✓ Fleksibel   ✓ Hemat biaya
🕸️

Multi-Cloud

Layanan dari dua penyedia atau lebih. Hindari vendor lock-in.

✓ Bebas lock-in   ✓ Redundansi
